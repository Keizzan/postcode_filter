Project Name: 
Postcodes filter 

Synopsis:
Program read ukpostcodes.csv file, validates its records and outputs dataframe with all postcodes within distance and html file containing map with highglighted desired radius

Application requirement:

For Developers
-----------------
    1. Create new file
    2. Import app.py.
    3. Initiate Postcode_dist class and provide desired distance(km) and postcode 
    4. Run your file.
